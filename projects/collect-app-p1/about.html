Project Overview

Country Explorer is a collection web application that allows users to search and explore country information using the Restful Countries API.

API Credits

This project uses the Restful Countries API to provide detailed information about countries including their capital, population, size, currency, flag, and more.

To access the API, you will need an API key. Please visit the Restful Countries API Documentation for more details and to obtain your token.

Supported Endpoints

  • getAllCountries(per_page)
  • getCountryByName(country)
  • getCountriesByContinent(continent, per_page)
  • getCountriesByPopulation(min, max, per_page)
  • getCountriesBySize(min, max, per_page)
  • getCountryByISO2(iso2) and getCountryByISO3(iso3)
